LAKUM ARTSPACE - QUICK START GUIDE
===================================

STEP 1: Import Database
------------------------
1. Open phpMyAdmin or MySQL command line
2. Import the file: database.sql
3. This will create the complete database with sample events

STEP 2: Update Database (If Upgrading)
---------------------------------------
If you already have a database and just need to add new columns:
Visit: http://localhost/Lakum_Basic/LAKUM/LAKUM/update_database.php

STEP 3: Admin Login
--------------------

STEP 4: View Calendar

FEATURES:
---------
✅ Calendar with month filtering
✅ Add/Edit events with modern forms
✅ Cover image and gallery management
✅ Start and end date/time support
✅ Delete gallery images in edit form (hover to see X button)
✅ Event detail page shows all date/time info
✅ Location display on event details
✅ Automatic event status updates (upcoming/past)

EVENT DETAIL PAGE:
------------------
Shows:
- Event title and cover image
- Start date and time
- End date and time (if different from start)
- Location with icon
- Full description
- Gallery slider

ADMIN FEATURES:
---------------
Add Event:
- Modern form with cover image preview
- Start/End date and time fields
- Location field
- Gallery upload with drag & drop
- Character counter for description

Edit Event:
- Same modern design as Add Event
- Existing gallery images with delete buttons (hover to see X)
- Time fields properly populated
- All fields editable

FILES:
------
- database.sql - Complete database schema (import this)
- update_database.php - Automated database updater
- Calender.php - Main calendar page
- event-detail.php - Event details page (updated)
- admin/ - Admin panel files
